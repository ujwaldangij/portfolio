# Ujwal Dangi Portfolio

Fast, static and dynamic JavaScript-powered portfolio for a Senior Software Engineer profile.

## What changed

- Java Spring Boot first positioning
- Removed old mixed-stack headline branding
- Faster site: removed heavy animation libraries and external font/CDN dependency
- All main content is managed from `js/main.js` inside `portfolioData`
- Experience years and footer year update automatically from JavaScript
- Resume, GitHub, LinkedIn and email links are controlled from one place

## Edit dynamic data

Open:

```text
js/main.js
```

Update only the `portfolioData` object for name, role, links, skills, experience, projects and resume path.
